Steps to run the FrontEnd Program.

Step 1->Run the index.html file.
Step 2->Index.html contains a search bar and a search button.
Step 3->Entering country name and clicking the search button,
        it makes an API call using fetch API.
Step 4->Then it displays the flag and country name as a card in same html page.
Step 5->on clicking the card  it redirects to the details.html file
        where the country's details are shown. 
Step 6->details.html has a back button to go back to the index.html page again.
